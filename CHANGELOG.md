# 📋 更新日志

## [1.1.0] - 2025-02-25

### ✨ 新增
- 扣费周期选择（月付 / 季付 / 年付）
- 年付、季付订阅自动折算月均费用
- 表单金额标签根据周期动态切换

### 🔧 优化
- 仪表板、统计页均使用折算后的月均金额
- 支出排行按月均金额排序

## [1.0.0] - 2025-02-25

### 🎉 首次发布
- 仪表板：月度总支出、活跃订阅数、年度预估
- 即将扣费：按日期排序显示近期扣费项
- 订阅管理：添加、编辑、暂停、删除
- 分类筛选：娱乐、工具、生活、通讯、健身、教育
- 支出统计：月度、日均、年度数据 + 排行
- PWA 支持：可添加到手机桌面，离线可用
- 数据本地存储（IndexedDB），隐私安全
- 部署到 GitHub Pages，自动化 CI/CD
