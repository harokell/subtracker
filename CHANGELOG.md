# 📋 更新日志

所有重要的更改都会记录在这里。

---

## [1.2.0] - 2025-02-25

### ✨ 新增
- 首次扣费日期选择器（替代原来的扣费日下拉）
- 支持记录已有会员的开通日期
- 系统根据开通日期 + 扣费周期自动计算下次扣费时间
- 自动推算上次扣费时间，订阅列表显示「上次→下次」时间线
- 即将扣费、订阅列表均显示精确的下次扣费日期

### 🔧 优化
- 年付/季付订阅现在能准确显示扣费月份和日期
- 表单添加提示文字，引导用户填写正确日期

---

## [1.1.0] - 2025-02-25

### ✨ 新增
- 扣费周期选择（月付 / 季付 / 年付）
- 年付、季付订阅自动折算月均费用
- 表单金额标签根据周期动态切换

### 🔧 优化
- 仪表板、统计页均使用折算后的月均金额
- 支出排行按月均金额排序

---

## [1.0.0] - 2025-02-25

### 🎉 首次发布
- 仪表板：月度总支出、活跃订阅数、年度预估
- 即将扣费：按日期排序显示近期扣费项
- 订阅管理：添加、编辑、暂停、删除
- 分类筛选：娱乐、工具、生活、通讯、健身、教育
- 支出统计：月度、日均、年度数据 + 排行
- PWA 支持：可添加到手机桌面，离线可用
- 数据本地存储（IndexedDB），隐私安全
- 部署到 GitHub Pages，自动化 CI/CD
